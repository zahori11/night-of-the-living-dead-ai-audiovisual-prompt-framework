# 🖼️ Visual Resources

Welcome to the Art and Postproduction Department! 🎨

This folder acts as our media archive (Media Pool). All graphic materials, keyframes, and generated image sequences that make up the film are stored here before final editing.

## 📁 Directory Contents

* **`/frames`**: Contains extracted keyframes from the original footage at a rate of **1 frame per second (1 FPS)**. These master images have gone through the stylization process (1-bit pixel art via Seedream/Freepik) and serve as the visual "pillars" (keyframes) used to guide AI animation or create retro *animatic* effects.

* **`Graphic Assets`** (e.g., CRT / Arcade Frames): Static set-design elements generated by AI (such as retro televisions or arcade machines) used as masks or overlays in the editing suite to frame the final footage.

## ⚙️ Postproduction Notes

To maintain the retro **"Game Boy Noir"** aesthetic, remember that any scaling applied to these images in your editing software must use the **"Nearest Neighbor"** (pixel replication) interpolation method or **"Hard Edges"**.
